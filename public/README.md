# Public Assets

This directory contains static assets that are served from the root URL.

## Required Image

### brand-banner.png
- **Purpose**: Open Graph image for social media sharing (Facebook, Twitter, LinkedIn, etc.)
- **Recommended Size**: 1200 x 630 pixels (1.91:1 aspect ratio)
- **File Location**: Place the image file directly in this directory as `brand-banner.png`
- **Usage**: Referenced in `app/layout.tsx` for Open Graph metadata

The image should represent your brand and work well as a social media preview card.
